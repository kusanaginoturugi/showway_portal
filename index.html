運用ツール Basic認証

Basic認証で保護されたサイト。死活判定は「HTTP応答が返るか (401を含む)」で行います。

ステータス凡例

  • UP: 応答あり (200 / 301 / 401 等)
  • DOWN: 到達不能 / ネットワークエラー
  • CHECKING: 確認中

ブラウザのCORS制約により、レスポンスボディもステータスコードも読めません。 そのため「応答が返ってきたか否か」のみで判定しています。Basic認証サイトでも 401 が返れば UP と判断されます。

注意: Cloudflare 等の CDN/プロキシ経由のサイトでは、オリジンサーバが 落ちていても CDN 側がエラーページ (520〜526 系) を返すため、本ページでは UP と表示されます。あくまで「エッジまでは到達できる」状態を示すものとお考えください。 厳密なオリジン死活は Grafana 等の外部監視で確認してください。
